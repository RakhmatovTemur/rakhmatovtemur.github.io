# Temurbek Rakhmatov - Personal Academic Website

This repository contains a simple static personal website designed for GitHub Pages hosting.

## Files

- `index.html` - single-page site content
- `styles.css` - styling and responsive layout
- `script.js` - menu interactions and subtle scroll animations
- `assets/images/profile.jpg` - profile image
- `assets/docs/CV_Rakhmatov_Temurbek.docx` - downloadable CV

## Publish on `rakhmatovtemur.github.io`

1. Create a GitHub repository named exactly `rakhmatovtemur.github.io`.
2. Add this project to that repository and push to the `main` branch.
3. In GitHub, open `Settings -> Pages` and ensure source is `Deploy from a branch`, branch `main`, folder `/ (root)`.
4. Your site will be available at `https://rakhmatovtemur.github.io/`.

## Notes

- This site is intentionally simple and optimized for academic use: clear sections, publication links, and contact details.
- It is pure HTML/CSS/JS, so it works natively with GitHub Pages.
